
# Sales Dashboard Streamlit

## Deskripsi
Aplikasi untuk analisis data penjualan menggunakan Streamlit, Pandas, dan SQLite.

## Fitur
- Upload file CSV/Excel
- Data cleaning otomatis
- Simpan ke database SQLite
- Visualisasi penjualan

## Tools
- Python
- Pandas
- Streamlit
- SQLite

## Cara Menjalankan
streamlit run app.py
